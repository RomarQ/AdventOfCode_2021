# Advent of Code 2021 | Solutions

The exercises can be found at https://adventofcode.com/2021

#### RUN javascript solution:

```sh
node e.js
```

#### RUN golang solution:

```sh
go run e.go
```

#### RUN Ocaml solution:

```sh
ocaml e.ml
```

#### RUN C solution:

```sh
gcc e.c && ./a.out
```
